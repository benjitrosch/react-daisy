## Quick Start

```bash
npm install react-daisy
```

Then import react-daisy components within your component files:

```js
import React from 'react'
import { Button } from '@levco/breezy'

export default (props) => {
  return (
    <Button color="primary">
      Click me!
    </Button>
  )
}
```
